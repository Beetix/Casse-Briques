# Casse-Briques

Un jeu de casse-briques (Breakout) jouable dans le navigateur — HTML5 Canvas + JavaScript vanilla, sans étape de build.

## Lancer le jeu

Ouvre simplement `index.html` dans un navigateur, ou sers le dossier :

```bash
python3 -m http.server 8000
# puis ouvre http://localhost:8000
```

## Stack

- HTML5 Canvas
- JavaScript vanilla (aucune dépendance)
